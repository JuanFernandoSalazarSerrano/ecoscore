====================================================================
4) SCORE COLOR LOGIC (DATA VISUAL FEEDBACK)
====================================================================
From app logic (Angular + React equivalent):
- score >= 80  -> emerald tone (text-emerald-500)
- score >= 60  -> lime tone (text-lime-500)
- score >= 40  -> amber tone (text-amber-500)
- else         -> red tone (text-red-500)

This provides immediate status semantics:
- Green/lime = healthy
- Amber = caution
- Red = poor

The visual stack uses:
